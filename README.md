Layout Website

- il progetto sarà pubblicato su github in una repository

- bootstrap lato frontend installato con npm

- Google font

- Fontawesome

- file style.css personale

- header comune per tutte le pagine riutilizzabile

- footer comune per tutte le pagine riutilizzabile

- ogni pagina scelta dagli elementi nella navbar(chi siamo, prodotti, contatti)
    porterà ad un file dedicato php dove avro' header e footer importati
    e una propria sezione della pagina

- nella navbar un bottone ci poterà ad una pagina di /login,
    nell aquale sarà prsente un form per poter eseguire l accesso (simulato)
    tramite l inserimento di un user e password
    se l accesso sarà effetuato correttamente,
    si etrerà in una pagina "protetta" -> dashboard.php
    se invece l utente sbaglia l accesso, verrà reindirizzato ad una pagina di non accesso.
